# basic.electrical.calculator
Calculadora básica que se rige por la "ley de ohm" y maneja ohms, ampers, volts y wats como magnitudes.
este es mi primer proyecto y se encuentra en desarrollo por lo que su uso practico esta en aprender intentando 
mejorarlo más que usarlo para lo que se supone que es. Para cada verción habra un README que explique los cambios
y uso particular.
